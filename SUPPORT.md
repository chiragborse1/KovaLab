# Support

Kova is beta software. The fastest support path is the community Discord:

https://discord.gg/uT9ETzpaHT

Use GitHub issues for reproducible bugs and focused feature requests:

https://github.com/chiragborse1/KovaLab/issues/new/choose

## Before Asking

Please include:

- Kova version or commit SHA.
- Install method.
- OS and Node version.
- The command you ran.
- The relevant error or log output with secrets removed.
- Whether this happens on a fresh setup or only an existing Kova home.

Useful commands:

```bash
kova status
kova doctor
kova security audit --deep
```

## Security Issues

Do not post security vulnerabilities in public issues or Discord.
Use the private process in `SECURITY.md`.
